## Deployment

Each subdirectory is a standalone Vite + React project with its own `netlify.toml`.
To deploy a project to Netlify:
1. Connect this repo in Netlify → **Add new site → Import an existing project**
2. Set the **Base directory** to the project subfolder (e.g. `nrma-agentic-fi`)
3. Build command and publish directory are defined in each project's `netlify.toml`
